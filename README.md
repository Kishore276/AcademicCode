# HackerRank Clone Frontend

This project is a React-based frontend for a HackerRank-like coding platform. It includes screens for Home, Challenges, Challenge Details, Login/Signup, and Profile.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Features
- Home screen
- Challenges list
- Challenge details
- User authentication (Login/Signup)
- User profile

## Tech Stack
- React
- Vite
- Modern UI libraries (add as needed)
